# Opening a single story

You can open individual stories using any interactive element in your app (banners, buttons, pop-ups) or push notifications.

All of our SDKs contain a function called showSingleStory (the name of the function differs between SDKs).

All you need to do is call this function in the click handler and pass it an ID or slug of a story that you want to show.

Follow these guides to learn more about showing a single story:

- [Android](/sdk-guides/android/single-story);
- [iOS](/sdk-guides/ios/single-story);
- [JS](/sdk-guides/js-sdk/single-story);
- [React](/sdk-guides/react-sdk/single-story);
