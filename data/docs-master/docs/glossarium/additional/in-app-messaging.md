# In-App Messaging

In-App Messages. These are popular visual formats on the market - bottomsheets, fullscreens, pop-ups, now available as new communication formats in InAppStory.

:::warning[Attention!]
This is an additional paid feature. Please contact the sales team to connect it to your project.
:::

There are two main ways to invoke In-App Messages.

1. Calling a message by id
2. Calling a message by event

Events serve as “labels” to determine the right set of messages to show to the user. Also, In-App Messages use the same targeting mechanism as Stories - tags and segments.

Follow these integration guides to add In-App Messages to your app:
