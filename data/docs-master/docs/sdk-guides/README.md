# Platform introduction

InAppStory provides a variety of SDKs for different apps and purposes. Currently, the following platforms are supported:

1. [Android](android/how-to-get-started.md)

2. [iOS (UIKit & SwiftUI)](ios/how-to-get-started.md)

3. [React Native](react-native/how-to-get-started.md)

4. [JavaScript](js-sdk/how-to-get-started.md)

5. [React](react-sdk/how-to-get-started.md)

6. [Flutter](flutter/how-to-get-started.md)

For UGC guides look [here](/ugc-guides/).
